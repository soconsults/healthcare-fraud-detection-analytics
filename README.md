# Healthcare Fraud Detection Analytics System

## Overview

This project demonstrates a production-grade healthcare claims fraud detection system using SQL, Python, and Power BI.

Designed to identify:

- Fraudulent providers
- Suspicious claims patterns
- High-risk provider behavior
- Claims anomalies

This project reflects real-world healthcare insurance analytics workflows.

---

## Business Problem

Healthcare fraud accounts for billions in losses annually. This system enables early fraud detection using data analytics techniques.

---

## Key Features

- Fraud detection SQL queries
- Provider risk scoring model
- Suspicious claims identification
- Executive dashboard-ready outputs

---

## Tools Used

- SQL
- Python (Pandas)
- Power BI
- Excel
